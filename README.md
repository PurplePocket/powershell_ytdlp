# powershell_ytdlp
Small script to download videos, music, and playlists with ytdlp

## Requirements

Supposedly nothing 

## Usage 

Right click and "execute with powershell"

Will: 
- create  C:\Youtube folder
- download and copy yt-dlp (download from youtube), magic (webm to jpg thumbnails) and ffmpeg (youtube music)
- prompt you a url, then how to use it
